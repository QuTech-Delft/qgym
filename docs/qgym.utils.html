class qgym.utils.RandomCircuitGenerator(n_qubits, max_gates, rng=None)[source]

Bases: object

Generates random circuits in the form of a List with Gate objects.

Example circuit:
>>> circuit = [Gate("prep", 0,0), Gate("prep", 1,1), Gate("cnot", 0,1)]
__init__(n_qubits, max_gates, rng=None)[source]

Initialize the RandomCircuitGenerator.

Parameters:

  • n_qubits (int) – Number of qubits of the circuit.

  • max_gates (int) – Maximum number of gates that a circuit may have.

  • rng (Optional[Generator]) – Optional random number generator.

__repr__()[source]

Create a string representation of the RandomCircuitGenerator.

generate_circuit(n_gates='random', mode='default')[source]

Generate a random quantum circuit.

Parameters:

  • n_gates (str | int) – If “random”, then a circuit of random length will be made. If an int is given, a circuit of length min(n_gates, max_gates) will be made.

  • mode (str) – If mode is “default”, a circuit will be generated containing the ‘prep’, ‘x’, ‘y’, ‘z’, ‘cnot’ and ‘measure’ gates. If mode is “workshop”, a simpler circuit containing just ‘h’, ‘cnot’ and measure gates will be generated.

Return type:

list[Gate]

Returns:

A randomly generated quantum circuit.

property rng: Generator

Return the random number generator of this RandomCircuitGenerator.

If none is set yet, this will generate a new one, using numpy.random.default_rng.
